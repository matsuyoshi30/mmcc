# [WIP] matsuyoshi30's mini C Compiler

![ci](https://github.com/matsuyoshi30/mmcc/workflows/ci/badge.svg)

Based on [compilerbook](https://www.sigbus.info/compilerbook), and using Docker.

```
$ docker build -t compilerbook .

$ ./docker-run
```

## Build

```
$ make
```

## Test

```
$ make test
```
